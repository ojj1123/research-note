- [monkey patch](https://en.wikipedia.org/wiki/Monkey_patch) : 런타임에 코드의 기본 동작에 조작(추가, 변경, 억제)을 가하는 것
```js
// monkey patch 예시
// 기존 console.log의 행위를 변형하고 있음
let log = console.log
console.log = function() {
	log.apply(console, [(new Date()).toString()].concat(arguments));
}
```
